## 📖 Cấu trúc thư mục & Nội dung khóa học

Dưới đây là cấu trúc các bài học và dự án được tổ chức theo từng thư mục.

-   **/bai-01-gioi-thieu-nodejs/**: Giới thiệu tổng quan về NodeJS, NPM, và các module cơ bản (path, fs).
-   **/bai-02-http-module/**: Xây dựng web server đầu tiên chỉ bằng module `http` sẵn có.
-   **/bai-03-expressjs-co-ban/**: Bắt đầu với framework Express.js, tìm hiểu về routing và middleware.
-   **/bai-04-template-engine/**: Sử dụng Template Engine (Handlebars/EJS) để render giao diện từ server.
-   **/bai-05-form-data-method/**: Xử lý dữ liệu gửi lên từ form (GET, POST).
-   **/bai-06-ket-noi-mongodb/**: Kết nối và thực hiện các thao tác CRUD với cơ sở dữ liệu MongoDB bằng Mongoose.
-   **/bai-07-restful-api/**: Xây dựng bộ RESTful API hoàn chỉnh cho một ứng dụng Blog.
-   **/bai-08-authentication-jwt/**: Triển khai xác thực và phân quyền người dùng bằng JSON Web Tokens (JWT).
-   **/bai-09-file-upload/**: Xử lý chức năng tải lên file (hình ảnh, tài liệu) lên server.
-   **/project-blog-nodejs/**: **Dự án cuối khóa** - Xây dựng một trang blog hoàn chỉnh tổng hợp các kiến thức đã học.

## 🛠️ Hướng dẫn cài đặt và sử dụng

1.  Clone repository này về máy của bạn:
    ```bash
    git clone https://github.com/hungdt31/nodeJS-f8.git
    ```

2.  Di chuyển vào thư mục của bài học hoặc dự án bạn muốn chạy:
    ```bash
    cd ten-thu-muc-bai-hoc
    ```

3.  Cài đặt các package cần thiết:
    ```bash
    npm install
    ```

4.  Khởi động dự án:
    ```bash
    npm start
    ```
    Hoặc (tùy thuộc vào file package.json):
    ```bash
    nodemon index.js
    ```
